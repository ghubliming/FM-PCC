# AUDIT REPORT: DA v3 INFRASTRUCTURE STABILIZATION

**Date**: 2026-05-13
**Subject**: Path Conciseness, Rogue Folder Elimination, and Explorer Automation
**Status**: COMPLETE / STABLE

---

## 1. Root Cause Analysis (RCA)

### Issue A: Rogue `checkpoints/` creation
- **Origin**: `Slurm_Codes/sbatch/iMF/eval_imf.sh`
- **Mechanism**: The bash script was passing `--checkpoint-dir checkpoints` to the evaluation entry point. The system's `Parser` class automatically created this directory upon seeing the argument, even if the application logic did not explicitly save checkpoints during evaluation.
- **Impact**: Polluted the repository root on remote compute nodes.

### Issue B: Visualizer Discovery Failure
- **Origin**: `Data_Analysis/Visualizer/index.html`
- **Mechanism**: Modern browsers block `fetch()` calls to parent directories (e.g., `../analysis_results`) if the web server is rooted in a subdirectory (`Visualizer/`). Additionally, relying on a static `results_manifest.json` was fragile if the DA run was interrupted.
- **Impact**: The "QUICK_LIST" feature was non-functional, requiring manual path entry.

---

## 2. Technical Resolution

### Backend Cleanliness
- **Slurm Patch**: Removed the redundant `--checkpoint-dir` flag from all iMeanFlow (iMF) evaluation templates.
- **DA Preservation**: The `main_da_batch.py` script was preserved in its original state to maintain backward compatibility and avoid bloated "bridge" logic.

### Frontend Intelligence ("Zero-Manifest" Protocol)
- **HTML Parsing Engine**: Upgraded the `index.html` PyScript engine to parse the HTML directory listing generated by `python -m http.server`.
- **Regex Logic**: `re.findall(r'href="(batch_v3_[^/"]+)/?"', html)`
- **Benefit**: The UI now dynamically discovers new batches the moment they appear on disk, without requiring any manifest generation in the Python backend.

### UI/UX Refinement
- **CSS Reset**: Overrode PyScript's `core.css` centering logic for form inputs.
- **Alignment**: Standardized left-alignment for variant and candidate checkboxes, ensuring readability for long experiment identifiers.
